Open-source CLI tool and desktop GUI for creating standards-compliant Digital Cinema Packages. Supports SMPTE & Interop, encryption, KDM generation, JPEG 2000 encoding, subtitles, 3D, and Atmos.

View on GitHub Quick Start

Features

Everything you need to create, verify, and deliver Digital Cinema Packages.

📦

DCP Creation

Create complete DCPs from JPEG 2000 image sequences and WAV audio. SMPTE ST 429 and Interop standards.

🎬

JPEG 2000 Encoding

Encode DPX, TIFF, EXR, PNG to JPEG 2000 codestream via grok or OpenJPEG with configurable bitrate.

🎞️

Video Transcoding

Transcode ProRes, H.264, H.265 video to image sequences via ffmpeg for J2K encoding pipeline.

🔐

Encryption & KDM

AES-128 content encryption with SMPTE-compliant KDM generation. Batch KDMs for multiple screens.

✅

DCP Verification

Validate DCPs using integrated dcpdoctor. Check compliance with SMPTE standards and Bv2.1 recommendations.

🎨

Colour Conversion

Rec.709, P3-D65, P3-DCI to XYZ colour space conversion with proper gamma 2.6 transfer function.

📝

Subtitles

SMPTE and Interop XML subtitle packaging. SRT import and conversion. Subtitle burn-in for festivals.

🔊

Audio & Atmos

Multi-channel audio (5.1, 7.1), Dolby Atmos IAB support, loudness measurement (EBU R128 / ATSC A/85).

🎥

Stereoscopic 3D

Frame interleaving for stereoscopic DCP creation from separate left/right eye image sequences.

📋

Version Files (VF)

Create supplemental VF DCPs referencing Original Version reels. Efficient distribution of alternate versions.

💾

Copy to Drive

Write DCPs to USB/CRU drives with post-copy hash verification for reliable delivery to cinemas.

🖥️

Desktop GUI

Tauri 2 desktop app with dark theme, drag-and-drop, job queue, and all CLI features in a visual interface.

Quick Start

Build from source in three steps.

Prerequisites

RequiredCMake ≥ 3.25
RequiredC++23 compiler
Requiredlibxml2
RequiredOpenSSL
RequiredXerces-C
Optionalffmpeg (transcoding)
Optionalgrok (fast J2K encoding)
OptionalNode.js + Rust (GUI)
1

Clone

git clone --recurse-submodules https://github.com/DcpDoctor/dcpwizard.git

2

Configure & Build

cmake -B build -G Ninja -DCMAKE_BUILD_TYPE=Release && cmake --build build --parallel

3

Test

cd build && ctest --output-on-failure

CLI Usage

Powerful command-line interface for scripting and automation.

# Create a DCP from J2K image sequence + WAV audio dcpwizard create --title "My Feature Film" --video ./j2k --audio ./audio.wav --output ./dcp # Encode image sequence to JPEG 2000 dcpwizard encode --input ./dpx --output ./j2k --bandwidth 250 --encoder grok # Transcode video to image sequence dcpwizard transcode --input movie.mov --output ./sequence # Verify an existing DCP dcpwizard verify ./my_dcp # Generate KDM for encrypted DCP dcpwizard kdm --dcp ./my_dcp --cert screen.pem --output kdm.xml # Copy DCP to cinema drive with hash verification dcpwizard copy --src ./my_dcp --dst /mnt/cru # Start REST API for headless operation dcpwizard serve --port 8080

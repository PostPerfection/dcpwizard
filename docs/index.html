Features

Everything you need to create, verify, and deliver Digital Cinema Packages.

📦

DCP Creation

OV DCPs, SMPTE & Interop. 2K/4K, frame rates 24–60 fps, HBR up to 500 Mbps. CPL / PKL / ASSETMAP / VOLINDEX generation with SHA-1 hashing.

🎬

JPEG 2000 Encoding

Encode DPX, TIFF, EXR, and PNG to J2K with OpenJPEG on the create video path; the streaming pipeline uses Grok.

🎞️

Video Import & Transcode

Import QuickTime, MP4, MXF, AVI, MKV, MJ2. Transcode ProRes, H.264, H.265, DNxHR via ffmpeg. Scale, crop, letterbox, pillarbox.

🔐

Encryption & KDM

AES-128 essence encryption. Signed SMPTE (ST 430-1/430-3) KDM generation carrying the DCP's image and audio content keys. Batch KDM, DKDM re-wrap, configurable validity periods and formulations (modified-transitional-1, DCI-any, DCI-specific).

✅

Quality Control

Integrated dcpdoctor for Bv2.1 compliance. HTML QC reports. Hash verification, XML schema validation. All standard markers.

🎨

Colour Conversion

Rec.709, P3-D65, P3-DCI → XYZ with gamma 2.6 transfer function. Correct colour pipeline for cinema projection.

📝

Subtitles

SRT → SMPTE / Interop subtitle XML conversion. Multilingual (RFC 5646). Burn-in for festivals.

🔊

Audio

PCM audio wrapping (48 kHz). Loudness measurement (EBU R128 / ATSC A/85). WAV input.

📤

Delivery

Copy to cinema drive with hash verification. Frame extraction.

⚡

Automation

Watch folder mode. Job queue with progress and cancellation. REST API. Docker container for CI/CD. Shell completions.

🖥️

Desktop GUI

Tauri 2 desktop app with dark theme. Drag-and-drop import, timeline scrubber with timecode, SRT→SMPTE subtitle panel, burn-in, keyboard shortcuts, and desktop notifications.

▶️

Preview & Playback

Frame-accurate DCP playback and frame extraction. Thumbnail generation. Visual QC before delivery.

📥

Camera Ingest

Direct ingest from ARRI, RED, Sony, Canon, and Blackmagic RAW camera originals into the DCP pipeline.

🎬

EDL/AAF Conform

Import CMX 3600 EDL, AAF, and XML timelines. Auto-assemble reels from edit decisions.

🔒

Certificate Management

X.509 certificate generation for content encryption. Trust chain management. Screen certificate import.

🌈

Dolby Vision & HDR

Dolby Vision RPU injection, HDR10 and HLG metadata handling for next-generation cinema mastering.

📊

Analytics & Monitoring

Real-time job monitoring via REST API. Prometheus-compatible /metrics endpoint for Grafana integration. Job queue statistics and system health.

Quick Start

Build from source in three steps.

Prerequisites

RequiredRust toolchain (1.85+)
RequiredOpenSSL
RequiredXerces-C
Optionalffmpeg (transcoding)
Optionalgrok (fast J2K encoding)
OptionalNode.js 24 + pnpm 11 + Rust (GUI)
1

Clone

git clone --recurse-submodules https://github.com/PostPerfection/dcpwizard.git

2

Build

cd rust && cargo build --release

3

Test

cd rust && cargo test --workspace

CLI Usage

Powerful command-line interface for scripting and automation.

# Create a DCP from J2K image sequence + WAV audio dcpwizard create --title "My Feature Film" --video ./j2k --audio ./audio.wav --output ./dcp # Create from video file (full pipeline: decode → J2K → MXF → DCP) dcpwizard create --title "My Film" --video movie.mov --output ./dcp --encoder openjpeg # Create 4K trailer with custom bitrate dcpwizard create --title "Trailer" --video trailer.mov --output ./dcp --content-type TLR --fourk --video-bit-rate 500 # Encode image sequence to JPEG 2000 dcpwizard encode --input ./dpx --output ./j2k --bandwidth 250 --encoder grok # Verify an existing DCP (strict mode, HTML report) dcpwizard verify ./my_dcp --strict --output report.html # Quick verification (skip hash checks) dcpwizard verify ./my_dcp --no-hash-check --quiet # Generate KDM with validity period (signer cert/key required) dcpwizard kdm --cpl-id <uuid> --content-title "My Film" --cert screen.pem \ --signer-cert signer.pem --signer-key signer.key --keys ./secret/my_film.keys.json \ --output kdm.xml --valid-from now --valid-to "2 weeks" # KDM with formulation dcpwizard kdm --cpl-id <uuid> --content-title "My Film" --cert screen.pem \ --signer-cert signer.pem --signer-key signer.key --keys ./secret/my_film.keys.json \ --output kdm.xml --formulation dci-any # Copy DCP to cinema drive with hash verification dcpwizard copy --src ./my_dcp --dst /mnt/cru # Start REST API for headless operation dcpwizard serve --bind 127.0.0.1:8080 # Convert SRT subtitles to SMPTE DCP XML dcpwizard subtitle-convert --input subs.srt --output subs.xml --language en # Burn subtitles into video dcpwizard burn-in --input movie.mov --subtitles subs.srt --output burned.mov # Scale/crop to target DCI resolution dcpwizard convert --input movie.mov --output movie_2k.mov --target 2k-scope # Create DCDM intermediate (Digital Cinema Distribution Master) dcpwizard dcdm -i ./frames -o ./dcdm --colour-space rec709 # Colour space conversion dcpwizard colour -i ./rec709_frames -o ./p3_frames --source rec709 --target p3 # Import EDL/AAF/XML timeline for conforming dcpwizard conform -i timeline.edl --json # Camera raw ingest dcpwizard ingest -s /mnt/camera_card -o ./dpx_frames -f dpx # Extract a frame from MXF/video dcpwizard frame-extract -i video.mxf -f 100 -o frame.png # Inject Dolby Vision RPU dcpwizard dv-inject -i input.hevc -r rpu.bin -o output.hevc # Job queue management dcpwizard daemon dcpwizard batch list dcpwizard batch add -T create-dcp -p '{"title":"Film","video":"j2k","output":"dcp"}'

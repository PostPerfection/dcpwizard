Features

Everything you need to create, verify, and deliver Digital Cinema Packages.

📦

DCP Creation

OV and VF DCPs, multi-CPL timelines, SMPTE Bv2.1 & Interop. 2K/4K, HFR up to 120fps, HBR up to 500 Mbps. Reel splitting, markers, VOLINDEX.

🎬

JPEG 2000 Encoding

Encode DPX, TIFF, EXR, PNG, BMP to J2K via grok (GPU) or OpenJPEG. J2K transcoder for bitrate re-encoding. Configurable bandwidth.

🎞️

Video Import & Transcode

Import QuickTime, MP4, MXF, AVI, MKV, MJ2. Transcode ProRes, H.264, H.265, DNxHR via ffmpeg. Scale, crop, letterbox, pillarbox.

🔐

Encryption & KDM

AES-128 encryption. KDM generation (Interop, SMPTE, Atmos). Batch KDM, DKDM, time zones, trusted device lists, annotation schemes.

✅

Quality Control

Integrated dcpdoctor for Bv2.1 compliance. HTML QC reports. Hash verification, XML schema validation. All standard markers.

🎨

Colour Conversion

Rec.709, P3-D65, P3-DCI → XYZ with gamma 2.6 transfer function. Correct colour pipeline for cinema projection.

📝

Subtitles

SMPTE 428-7 CineCanvas & Interop XML. SRT import/conversion. Multilingual (RFC 5646). Burn-in for festivals.

🔊

Audio, Atmos & DTS:X

Mono to 7.1 channel audio. Dolby Atmos (IAB) and DTS:X immersive packaging. Loudness (EBU R128 / ATSC A/85). Channel mapping.

🎥

Stereoscopic 3D

Frame interleaving from separate L/R eye sequences. Full 3D DCP creation with proper MXF track packaging.

📤

Export & Delivery

Export DCP to ProRes, H.264, H.265, DNxHR, or image sequence. Copy to drive with hash verification. Frame extraction.

⚡

Automation

Watch folder mode. Job queue with progress and cancellation. REST API. Docker container for CI/CD. Shell completions.

🖥️

Desktop GUI

Tauri 2 desktop app with dark theme. All CLI features in a visual interface with drag-and-drop and real-time job monitoring.

▶️

Preview & Playback

Frame-accurate DCP playback and frame extraction. Thumbnail generation. Visual QC before delivery.

📥

Camera Ingest

Direct ingest from ARRI, RED, Sony, Canon, and Blackmagic RAW camera originals into the DCP pipeline.

🎬

EDL/AAF Conform

Import CMX 3600 EDL, AAF, and XML timelines. Auto-assemble reels from edit decisions.

🔒

Certificate Management

X.509 certificate generation for content encryption. Trust chain management. Screen certificate import.

🌈

Dolby Vision & HDR

Dolby Vision RPU injection, HDR10 and HLG metadata handling for next-generation cinema mastering.

📊

Analytics Dashboard

Real-time job monitoring, bitrate analytics, throughput charts, and system health metrics via Prometheus endpoint.

Quick Start

Build from source in three steps.

Prerequisites

RequiredCMake ≥ 3.25
RequiredC++23 compiler
Requiredlibxml2
RequiredOpenSSL
RequiredXerces-C
RequiredSQLite3
Optionalffmpeg (transcoding)
Optionalgrok (fast J2K encoding)
OptionalNode.js + Rust (GUI)
1

Clone

git clone --recurse-submodules https://github.com/DcpDoctor/dcpwizard.git

2

Configure & Build

cmake -B build -G Ninja -DCMAKE_BUILD_TYPE=Release && cmake --build build --parallel

3

Test

cd build && ctest --output-on-failure

CLI Usage

Powerful command-line interface for scripting and automation.

# Create a DCP from J2K image sequence + WAV audio dcpwizard create --title "My Feature Film" --video ./j2k --audio ./audio.wav --output ./dcp # Encode image sequence to JPEG 2000 dcpwizard encode --input ./dpx --output ./j2k --bandwidth 250 --encoder grok # Transcode video to image sequence dcpwizard transcode --input movie.mov --output ./sequence # Verify an existing DCP dcpwizard verify ./my_dcp # Generate KDM for encrypted DCP dcpwizard kdm --dcp ./my_dcp --cert screen.pem --output kdm.xml # Copy DCP to cinema drive with hash verification dcpwizard copy --src ./my_dcp --dst /mnt/cru # Start REST API for headless operation dcpwizard serve --port 8080
